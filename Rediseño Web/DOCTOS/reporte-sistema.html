Los 7 flujos que armé en n8n

Cada flujo es una automatización que corre en n8n instalado en mi propia PC, sin pagar servidor externo.

Flujo 01
Lead desde el sitio web
Cuando alguien llena el cotizador, el lead llega automáticamente a Google Sheets, Telegram y correo en menos de 5 segundos.
Google SheetsTelegramEmail
Flujo 02
Follow-up automático
A los 2, 5 y 10 días de recibir un lead sin respuesta, el sistema envía recordatorio por correo al prospecto.
Cron cada 6hEmail
Flujo 03
Lead desde Facebook Ads
Los leads que entran por Facebook Lead Ads se procesan igual que los del sitio web: Sheets + Telegram + Email.
FacebookGoogle Sheets
Flujo 04
Generar link de menú
Cuando firmo contrato con un cliente, lleno un formulario interno y el sistema genera su token único y link personalizado de selección de menú.

Servir formulario de menú
Cuando el cliente abre su link, el sistema valida su token, lee sus datos del Sheets y le muestra el formulario personalizado con las opciones de su paquete.

Operación diaria

Cómo entrego el link de menú a un cliente

Este es el proceso exacto que sigo cada vez que firmo contrato con un cliente.

1

Abro el formulario interno de generación de link

Entro a http://localhost:5678, busco el Flujo 4 y abro el formulario. También puedo entrar directamente al webhook del formulario desde n8n.

2

Lleno los datos del cliente

Nombre, email, teléfono, fecha del evento, tipo de evento y paquete contratado. El sistema genera un token único automáticamente (ej: LC-1746832901234-AB8K2J).

3

El sistema guarda el token en Google Sheets

En la hoja "Selecciones" queda registrado el cliente con status pendiente. El link generado se guarda también ahí.

4

Envío el link al cliente por WhatsApp o correo

El link tiene este formato:
https://n8n.lacabanaeventos.com/webhook/seleccion?token=LC-XXXX-XXXXXX
Es de un solo uso — una vez que el cliente selecciona su menú, el token se marca como completado y no puede usarse de nuevo.

5

El cliente elige su menú y confirma

El formulario le muestra las opciones correspondientes a su paquete (Clásico, Gourmet o Premium). Al confirmar, me llega notificación por Telegram y queda registrado en Sheets.

⚠️
El token de prueba TEST-ANGEL-001 siempre funciona para probar el sistema sin afectar datos reales. Nunca lo envíes a un cliente real.

Operación

Cómo arranco el sistema cada día

✅
El sistema se inicia con un solo clic desde el acceso directo en el escritorio: "INICIAR LA CABANA.bat"

El script hace automáticamente:

1

Verifica si n8n ya está corriendo

Si no está activo, lo inicia y espera a que esté listo.

2

Activa los flujos 5 y 6

Los flujos de selección de menú se activan automáticamente.

3

Verifica el Cloudflare Tunnel

Si el tunnel no está corriendo, lo inicia para que el sistema sea accesible desde internet.

⚡
Importante: La PC donde corre n8n debe estar encendida y con internet para que el sistema funcione. Si se apaga, los webhooks dejan de responder.
