Flujos 4, 5 y 6 - sistema de token

La Cabana Eventos: formulario de seleccion de menu para cliente que agenda un salon de eventos.

La idea en corto

El token es como una pulsera de entrada. Si la pulsera existe en Google Sheets y dice pendiente, el cliente puede abrir el formulario. Cuando lo manda, n8n cambia esa pulsera a completado y ya no sirve otra vez.

Tu llenas datos del eventoNombre, telefono, correo, fecha, tipo de evento y paquete.
↓
Flujo 4 fabrica el tokenEjemplo: LC-1770000000000-ABC123
↓
Sheets guarda status pendienteLa pestana importante se llama Selecciones.
↓
Cliente abre el linkhttps://n8n.lacabanaeventos.com/webhook/seleccion?token=TOKEN
↓
Flujo 5 revisa y muestra formularioSolo si el token sigue pendiente.
↓
Flujo 6 guarda la seleccionMenu, invitados, cocteleria, cerveza, dietas y notas.
↓
Sheets cambia a completadoEl mismo link queda cerrado.

Flujo 4: generar link

Webhook o formulario internoRecibe datos del cliente y evento.
↓
Code - Generar tokenUsar archivo n8n_flujo4_generar_token.js.
↓
IF - Datos validos?Si falta nombre, email, telefono, fecha, tipo o paquete, responde error.
↓
Google Sheets - Agregar filaGuarda en Selecciones con status pendiente.
↓
TelegramTe manda el link para copiarlo a WhatsApp.

Flujo 5: abrir formulario

Webhook GET /seleccionLee el token desde query.token.
↓
Sheets - Leer seleccionesBusca la fila del token.
↓
Code - Validar statustoken + status pendiente.
↓
ValidoResponder HTML del formulario.
No validoResponder "Este link ya no esta disponible".

Flujo 6: guardar seleccion

Webhook POST /seleccion-guardarRecibe el formulario del cliente.
↓
Sheets - Leer para validarVuelve a buscar el token. Esta segunda revision es obligatoria.
↓
IF - Token valido?Solo acepta si status sigue pendiente.
↓
ValidoActualizar la misma fila por token y poner status completado.
No validoRechazar intento duplicado.
↓
Email + TelegramConfirmacion al cliente y alerta interna.

Prueba final

Despues de pegar los codigos en n8n, la prueba debe hacerse asi:

1

Ejecuta Flujo 4 con cliente de prueba.

2

Confirma fila nueva en Selecciones con status=pendiente.

3

Abre el link generado y verifica que salga el formulario.

4

Envia una seleccion de menu.

5

Confirma que la misma fila queda en status=completado.

6

Abre el mismo link otra vez: debe decir que ya no esta disponible.
